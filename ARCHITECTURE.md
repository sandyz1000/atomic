# Architecture

The architecture documentation lives in the docs site under
[`docs/src/content/docs/architecture/`](docs/src/content/docs/architecture/):

- [Overview](docs/src/content/docs/architecture/overview.md) — crates, execution flow, key types
- [Execution Model](docs/src/content/docs/architecture/execution-model.md) — tasks, staged pipelines, version safety
- [Shuffle](docs/src/content/docs/architecture/shuffle.md) — how data moves between stages

Run the docs site locally with `cd docs && npm install && npm run dev`.
